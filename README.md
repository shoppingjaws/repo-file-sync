# repo-file-sync

A fast, efficient repository file synchronization tool built with Bun.

## Quick Start

1. **Install Bun** (if not already installed):
   ```bash
   curl -fsSL https://bun.sh/install | bash
   ```

2. **Install dependencies**:
   ```bash
   bun install
   ```

3. **Run the application**:
   ```bash
   bun start
   ```
